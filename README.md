# spb-boat-rental-landing

Одностраничный лендинг: **аренда катера без капитана в Санкт-Петербурге**.
Самостоятельные прогулки по Неве, каналам и Финскому заливу после инструктажа.

## Стек

- [Next.js](https://nextjs.org) (App Router)
- TypeScript
- Tailwind CSS
- npm

## Блоки главной страницы

Header · Hero · Преимущества · Для кого · Катера · Маршруты · Как проходит
аренда · Безопасность · Цены и условия · Форма бронирования · FAQ · Финальный
CTA · Footer.

## Структура проекта

```
app/
  layout.tsx          # шрифт, мета, Header, Footer
  page.tsx            # сборка секций главной страницы
  globals.css         # тема Tailwind (светлая, водная)
  privacy/            # политика конфиденциальности
  terms/              # пользовательское соглашение
  rental-rules/       # правила аренды
  cookie/             # использование cookie
components/           # секции лендинга и UI-компоненты
data/                 # контент: катера, маршруты, FAQ, преимущества и т.д.
public/images/        # SVG-визуалы (hero, катера)
```

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Форма бронирования

Форма имеет клиентскую валидацию. При успешной отправке показывается сообщение:

> Заявка принята в тестовом режиме. Для боевого запуска подключите Telegram,
> CRM или почту в API-обработчике.

Для боевого режима добавьте серверный обработчик (Route Handler в `app/api/...`)
и отправку заявок в Telegram / CRM / на почту.

## Важно по контенту

- Не используются выдуманные точные цены — условия уточняет менеджер.
- Не утверждается «без прав», если это не подтверждено.
- Нет обещаний «100% безопасности».
- Финальные условия аренды подтверждает менеджер перед бронированием.
